关于图片取模工具

程序文件格式为exe格式，使用时若需要图片帧拆分，可以选择网上的免费的拆分工具。
作者这里采用的是https://uutool.cn/gif2img/。
后期使用可自行选择其他的图片拆分工具，例如百度给出的FlexClip、ruseo等在线工具。
